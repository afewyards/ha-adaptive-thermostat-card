# Adaptive Thermostat Card

Climate card for Home Assistant, based on [lovelace-mushroom](https://github.com/piitaya/lovelace-mushroom) by piitaya.

## Installation

1. Download the latest release
2. Copy to your `www` folder
3. Add resource in Lovelace

## Usage

```yaml
type: custom:mushroom-climate-card
entity: climate.your_entity
```
